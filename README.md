# bot

A twitch bot written in crystal

## Requirments

- [crystal][crystal-lang]

## Installation

```
$ crystal deps
$ crystal src/bot/bot.cr
```

### Credentials

```
$ mkdir ./config
$ cp -r ./.config ./config
$ $EDITOR ./config/credentials.yml
```
